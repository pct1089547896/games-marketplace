HERO SECTION MEDIA PRESERVATION - FIX COMPLETE
================================================================================

PROBLEM FIXED:
--------------
Hero section background images and videos were being hidden or affected by 
theme CSS overrides. When users applied themes, hero media became invisible
or distorted.

ROOT CAUSE:
-----------
The CSS theme overrides were too broad and affected ALL elements with classes
like .bg-black, .text-white, including hero section backgrounds and videos.

SOLUTION:
---------
Modified CSS override rules to EXCLUDE hero section media elements:

1. Added :not() selectors to skip positioned elements (hero backgrounds)
2. Preserved video[autoplay] elements (hero videos)
3. Preserved elements with inline backgroundImage styles
4. Preserved gradient backgrounds (bg-gradient-*)
5. Preserved text with inline color styles

-----------------------
✅ Hero text without inline styles
✅ Hero buttons and links
✅ Hero interactive elements
✅ Rest of page content (cards, navigation, etc.)

WHAT STAYS UNCHANGED:
---------------------
✅ Hero background videos
✅ Hero background images
