PROBLEM FIXED:
--------------
Themes were loading in admin panel but not applying to the frontend.
Users could click "Apply Theme" but the website colors wouldn't change.

ROOT CAUSE:
-----------
CSS variables were being set, but the frontend components used hardcoded 
Tailwind classes (bg-white, text-black, etc.) that don't automatically 
respond to CSS variable changes.

SOLUTION:
---------
1. Modified ThemeContext to set a data attribute on body element
2. Created CSS cascade system that overrides Tailwind classes
3. Added console logging for debugging
